# 疫情/健康提醒文章质量评分标准

> 总分 40 分，四个维度各 10 分。
> 32 分及以上为优秀，24-31 分为合格，24 分以下需修改。

---

## 维度一：结构完整性（10 分）

基于蓝图要求的 5 个标准段落进行评估。

| 检查项 | 分值 | 评分标准 |
|--------|------|---------|
| 包含"速览"段且 1-2 句话概括核心信息 | 2 分 | 精炼且有冲击力 = 2，有但略冗长 = 1，无 = 0 |
| 包含"当前形势"段且有数据支撑 | 2.5 分 | 数据完整、有对比参照 = 2.5，有数据但缺对比 = 1.5，无数据 = 0 |
| 包含"风险提示"段且区分高风险人群 | 2 分 | 有分级提示 = 2，仅泛泛提醒 = 1，无 = 0 |
| 包含"防护建议"段且分级、具体可执行 | 2 分 | 分级且可操作 = 2，有建议但笼统 = 1，无 = 0 |
| 包含"信息来源"段且标注数据出处 | 1.5 分 | 来源清晰 = 1.5，有但不完整 = 0.5，无 = 0 |

---

## 维度二：知识准确性（10 分）

| 检查项 | 分值 | 评分标准 |
|--------|------|---------|
| 所有数据标注"截至 XXXX 年 X 月 X 日" | 2.5 分 | 全部标注 = 2.5，部分遗漏 = 1，无标注 = 0 |
| 数据来自权威来源（疾控中心、卫健委等） | 2.5 分 | 全部权威 = 2.5，混有非权威 = 1，来源不明 = 0 |
| 风险评估客观，未夸大也未淡化 | 2 分 | 客观准确 = 2，有偏颇 = 1，严重失实 = 0 |
| 高风险人群描述与官方指引一致 | 1.5 分 | 完全一致 = 1.5，有遗漏 = 0.5，有错误 = 0 |
| 引用标注规范，每个数据有 {ref:N} | 1.5 分 | 规范完整 = 1.5，有遗漏 = 0.5，无引用 = 0 |

**扣分项**：
- 使用"暴发""失控""崩溃"等恐慌性词汇：扣 2 分
- 数据超过 30 天未标注"历史数据"：扣 1 分
- 引用非官方渠道的疫情数据：扣 3 分

---

## 维度三：可读性（10 分）

| 检查项 | 分值 | 评分标准 |
|--------|------|---------|
| 开头速览简洁有力，10 秒内传达核心 | 2 分 | 简洁有力 = 2，有但不够精炼 = 1，拖沓 = 0 |
| 数据有通俗翻译（类比/对比/聚焦） | 2.5 分 | 全部翻译 = 2.5，部分翻译 = 1.5，纯数字 = 0 |
| 防护建议以清单形式呈现，具体可执行 | 2.5 分 | 清单化且具体 = 2.5，有清单但笼统 = 1.5，文字堆砌 = 0 |
| 语气紧迫但冷静，未制造恐慌 | 2 分 | 平衡得当 = 2，偏恐慌 = 1，过于平淡 = 0.5 |
| 时间标记醒目，读者能感知时效性 | 1 分 | 醒目 = 1，有但不突出 = 0.5，缺失 = 0 |

---

## 维度四：平台适配（10 分）

| 检查项 | 分值 | 评分标准 |
|--------|------|---------|
| 标题 14-30 字，含核心关键词和时间标记 | 2 分 | 含时间标记 = 2，无时间标记但合规 = 1，标题党 = 0 |
| 字数在 400-800 字区间（速报可更短） | 2 分 | 符合 = 2，略超 = 1，严重偏离 = 0 |
| 防护建议用列表/清单格式 | 2 分 | 列表规范 = 2，基本清晰 = 1，混乱 = 0 |
| 数据用加粗或独立段落突出 | 2 分 | 关键数据突出 = 2，部分突出 = 1，未突出 = 0 |
| 无营销用语、无 HTML 标签 | 2 分 | 全部合规 = 2，有 1 处 = 1，多处 = 0 |

---

## 综合评分计算

```
总分 = 结构完整性 + 知识准确性 + 可读性 + 平台适配

32-40 分：优秀，可直接发布
24-31 分：合格，建议微调后发布
16-23 分：需修改，存在明显问题
0-15 分：需重写
```

## 疫情提醒特别扣分项

- **数据无时间标注**：每处扣 1 分（最高扣 3 分）
- **使用恐慌性词汇**：每处扣 1 分（最高扣 3 分）
- **防护建议不够具体**：如"注意防护""做好预防"等空话，扣 2 分
- **未区分高风险人群**：扣 1.5 分

## 一票否决项

以下任何一项出现，直接判定为"需重写"：
- 出现安全红线违规
- 疫情数据完全无来源标注
- 使用"失控""崩溃""医疗挤兑"等极端词汇
- 核心疫情数据明显错误
- 将疫情与特定地区/人群污名化关联
