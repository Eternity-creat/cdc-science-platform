# 谣言粉碎文章质量评分标准

> 总分 40 分，四个维度各 10 分。
> 32 分及以上为优秀，24-31 分为合格，24 分以下需修改。

---

## 维度一：结构完整性（10 分）

基于蓝图要求的 5 个标准段落进行评估。

| 检查项 | 分值 | 评分标准 |
|--------|------|---------|
| 包含"谣言引用"段且直接展示谣言原文 | 1.5 分 | 直接引用、交代传播背景 = 1.5，模糊提及 = 0.5，无 = 0 |
| 包含"真相速览"段且一句话给出判定 | 1.5 分 | 判定清晰果断 = 1.5，有但不够果断 = 0.5，无 = 0 |
| 包含"逐条拆解"段且每条有证据支撑 | 3 分 | 每条有证据 + 有"为什么信"分析 = 3，有证据但缺心理分析 = 2，证据薄弱 = 1，无 = 0 |
| 包含"辨别方法"段且给出可操作的检查步骤 | 2 分 | 3-5 条可操作方法 = 2，方法笼统 = 1，无 = 0 |
| 包含"核心要点"段且浓缩核心真相 | 2 分 | 精炼有力 = 2，有但冗长 = 1，无 = 0 |

**加分项**：拆解过程中体现了"谣言为什么有道理的外壳"的深层分析，可加 1 分（总分不超过 10 分）。

---

## 维度二：知识准确性（10 分）

| 检查项 | 分值 | 评分标准 |
|--------|------|---------|
| 判定结论正确，有权威证据支撑 | 3 分 | 结论正确且证据充分 = 3，结论正确但证据不够 = 2，结论有误 = 0 |
| 引用的研究/数据来自权威来源 | 2 分 | 全部权威 = 2，混有非权威 = 1，来源不明 = 0 |
| 在辟谣过程中未引入新的错误信息 | 2 分 | 无新问题 = 2，有 1 处存疑 = 1，有明显错误 = 0 |
| "为什么人们会信"的分析合理、有心理学依据 | 1.5 分 | 分析到位 = 1.5，有分析但浅 = 0.5，无分析 = 0 |
| 引用标注规范 | 1.5 分 | 规范完整 = 1.5，有遗漏 = 0.5，无引用 = 0 |

**扣分项**：
- 在辟谣中引入新的未经证实信息：扣 2 分
- 过度展开谣言内容（可能造成二次传播）：扣 1 分
- 对部分属实的谣言一刀切判定为"完全虚假"：扣 2 分

---

## 维度三：可读性（10 分）

| 检查项 | 分值 | 评分标准 |
|--------|------|---------|
| 开头引发"我见过这个"的共鸣 | 2 分 | 共鸣感强 = 2，有但弱 = 1，无共鸣 = 0 |
| 逐条拆解结构清晰，每条独立成段 | 2 分 | 结构清晰 = 2，基本清晰 = 1，混乱 = 0 |
| 语气果断但不嘲讽信谣者 | 2.5 分 | 果断且共情 = 2.5，果断但缺共情 = 1.5，嘲讽/居高临下 = 0 |
| "辨别方法"部分实用、可操作 | 2 分 | 读者能立即上手 = 2，方法偏抽象 = 1，不可操作 = 0 |
| 适度的幽默感，让文章不枯燥 | 1.5 分 | 幽默恰到好处 = 1.5，有尝试但生硬 = 0.5，完全无幽默 = 0 |

---

## 维度四：平台适配（10 分）

| 检查项 | 分值 | 评分标准 |
|--------|------|---------|
| 标题含辟谣关键词，未完整复述谣言 | 2 分 | 符合要求 = 2，标题含完整谣言 = 0.5，标题党 = 0 |
| 谣言原文用引用块（>）标记，与正文区分 | 2 分 | 格式规范 = 2，有区分但不规范 = 1，无区分 = 0 |
| 结论用加粗突出，一目了然 | 2 分 | 突出 = 2，有但不突出 = 1，缺失 = 0 |
| 字数在 600-1500 字区间 | 2 分 | 符合 = 2，略超 = 1，严重偏离 = 0 |
| 无营销用语、无 HTML 标签 | 2 分 | 全部合规 = 2，有 1 处 = 1，多处 = 0 |

---

## 综合评分计算

```
总分 = 结构完整性 + 知识准确性 + 可读性 + 平台适配

32-40 分：优秀，可直接发布
24-31 分：合格，建议微调后发布
16-23 分：需修改，存在明显问题
0-15 分：需重写
```

## 谣言粉碎特别扣分项

- **嘲讽信谣者**（如"这么明显的谣言都信"）：扣 3 分
- **标题中完整复述谣言**（标题本身成为谣言传播源）：扣 2 分
- **省略"Why"步骤**（不分析为什么人们会信）：扣 1.5 分
- **省略"How"步骤**（不教读者辨别方法）：扣 1.5 分
- **只辟谣不给证据**：扣 3 分

## 一票否决项

以下任何一项出现，直接判定为"需重写"：
- 出现安全红线违规
- 辟谣结论本身有误（把真相当谣言辟了）
- 在辟谣过程中引入新的伪科学信息
- 使用阴谋论框架（"境外势力造谣"）
- 全文无任何权威来源引用
