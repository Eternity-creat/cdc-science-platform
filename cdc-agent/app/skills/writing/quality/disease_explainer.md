# 疾病科普文章质量评分标准

> 总分 40 分，四个维度各 10 分。
> 32 分及以上为优秀，24-31 分为合格，24 分以下需修改。

---

## 维度一：结构完整性（10 分）

基于蓝图要求的 7 个标准段落进行评估。

| 检查项 | 分值 | 评分标准 |
|--------|------|---------|
| 包含"引入"段且开头有吸引力 | 1.5 分 | 有 = 1.5，有但平淡 = 0.5，无 = 0 |
| 包含"疾病定义"段且准确清晰 | 1.5 分 | 有且准确 = 1.5，有但含糊 = 0.5，无 = 0 |
| 包含"传播途径"段且有生活化说明 | 1.5 分 | 有且场景化 = 1.5，有但教科书式 = 0.5，无 = 0 |
| 包含"症状表现"段且有鉴别指引 | 1.5 分 | 有且有对比 = 1.5，有但仅罗列 = 0.5，无 = 0 |
| 包含"预防措施"段且为核心段落 | 1.5 分 | 有且字数最多、可操作性强 = 1.5，有但笼统 = 0.5，无 = 0 |
| 包含"就医指引"段且含"具体诊疗请遵医嘱" | 1.5 分 | 有且完整 = 1.5，有但缺"遵医嘱" = 0.5，无 = 0 |
| 包含"总结"段且浓缩核心信息 | 1 分 | 有且精炼 = 1，有但冗长 = 0.5，无 = 0 |

**加分项**：段落间逻辑连贯、过渡自然，可加 0.5 分（总分不超过 10 分）。

---

## 维度二：知识准确性（10 分）

| 检查项 | 分值 | 评分标准 |
|--------|------|---------|
| 疾病定义和分类准确，与最新指南一致 | 2 分 | 完全准确 = 2，有细微偏差 = 1，明显错误 = 0 |
| 传播途径描述科学，无遗漏主要途径 | 2 分 | 完整准确 = 2，遗漏次要途径 = 1.5，遗漏主要途径 = 0 |
| 症状描述与临床实际一致，含重症预警信号 | 2 分 | 完整且含重症信号 = 2，有症状但缺重症信号 = 1，不准确 = 0 |
| 预防措施有科学依据，未包含未经验证的方法 | 2 分 | 全部有依据 = 2，含 1 条存疑建议 = 1，含伪科学 = 0 |
| 引用标注规范，每个关键事实有 {ref:N}，来源权威 | 2 分 | 规范完整 = 2，有遗漏 = 1，无引用 = 0 |

**扣分项**：
- 使用非权威来源（如自媒体、论坛）：扣 2 分
- 数据无时间标注：每处扣 0.5 分
- 出现已辟谣的说法：扣 3 分

---

## 维度三：可读性（10 分）

| 检查项 | 分值 | 评分标准 |
|--------|------|---------|
| 开头在 3 句话内建立"跟我有关"的感觉 | 2 分 | 强力钩子 = 2，平淡开头 = 1，无吸引力 = 0 |
| 每段不超过 4 行，段间有空行 | 1.5 分 | 全部符合 = 1.5，个别超标 = 1，多处超标 = 0 |
| 专业术语有通俗解释 | 2 分 | 全部解释 = 2，解释 1-2 个 = 1，未解释 = 0 |
| 数据有通俗翻译（类比/对比/聚焦） | 2 分 | 全部翻译 = 2，部分翻译 = 1，纯数字罗列 = 0 |
| 整体语气专业温暖，有同理心 | 2.5 分 | 语气得当 = 2.5，略生硬 = 1.5，居高临下 = 0 |

---

## 维度四：平台适配（10 分）

| 检查项 | 分值 | 评分标准 |
|--------|------|---------|
| 标题 14-30 字，含核心关键词，无标题党 | 2 分 | 完全符合 = 2，字数略超 = 1，标题党 = 0 |
| 摘要 80-120 字，概括文章核心价值 | 1.5 分 | 完全符合 = 1.5，过长或过短 = 0.5，无 = 0 |
| 小标题使用中文数字或阿拉伯数字编号 | 1.5 分 | 规范使用 = 1.5，格式不统一 = 0.5，无 = 0 |
| 有配图建议且密度合理（每 400-600 字 1 张） | 1.5 分 | 密度合理 = 1.5，偏少或偏多 = 0.5，无 = 0 |
| 字数在 600-1500 字区间 | 1.5 分 | 符合 = 1.5，略超 = 1，严重偏离 = 0 |
| 无营销用语、无 HTML 标签、无脚注 | 2 分 | 全部合规 = 2，有 1 处违规 = 1，多处违规 = 0 |

---

## 综合评分计算

```
总分 = 结构完整性 + 知识准确性 + 可读性 + 平台适配

32-40 分：优秀，可直接发布
24-31 分：合格，建议微调后发布
16-23 分：需修改，存在明显问题
0-15 分：需重写
```

## 一票否决项

以下任何一项出现，直接判定为"需重写"：
- 出现安全红线违规（参见 universal_rules.md 第三节）
- 包含恐慌性语言
- 推荐具体药物品牌或医院
- 核心医学信息明显错误
- 无任何引用标注
