# 季节性健康提示文章质量评分标准

> 总分 40 分，四个维度各 10 分。
> 32 分及以上为优秀，24-31 分为合格，24 分以下需修改。

---

## 维度一：结构完整性（10 分）

基于蓝图要求的 5 个标准段落进行评估。

| 检查项 | 分值 | 评分标准 |
|--------|------|---------|
| 包含"季节特点"段且有感官描写、建立与健康的关联 | 2 分 | 有感官描写 + 关联健康 = 2，有描写但未关联 = 1，无 = 0 |
| 包含"高发疾病"段且列出 3-5 种，每种有简要说明 | 2.5 分 | 3-5 种且有说明 = 2.5，列出但说明不足 = 1.5，少于 3 种 = 1，无 = 0 |
| 包含"预防要点"段且以清单/检查表形式呈现 | 2.5 分 | 清单化且具体 = 2.5，有但非清单化 = 1.5，笼统 = 0.5，无 = 0 |
| 包含"生活建议"段且涵盖多维度（饮食/运动/睡眠等） | 2 分 | 多维度且具体 = 2，仅 1-2 个维度 = 1，无 = 0 |
| 包含"总结"段且温暖收束 | 1 分 | 温暖精炼 = 1，有但平淡 = 0.5，无 = 0 |

---

## 维度二：知识准确性（10 分）

| 检查项 | 分值 | 评分标准 |
|--------|------|---------|
| 高发疾病确实为该季节高发病，与监测数据一致 | 2.5 分 | 完全准确 = 2.5，有 1 种不准确 = 1，多种不准确 = 0 |
| 预防措施有科学依据，无伪养生内容 | 2.5 分 | 全部有依据 = 2.5，含 1 条存疑 = 1.5，含伪科学 = 0 |
| 生活建议科学合理，未推荐特定保健品/产品 | 2 分 | 科学合理 = 2，含产品推荐 = 0.5，伪科学 = 0 |
| 季节特点描述准确（气温、湿度、花粉等） | 1.5 分 | 准确 = 1.5，大致准确 = 1，明显错误 = 0 |
| 引用标注规范 | 1.5 分 | 规范完整 = 1.5，有遗漏 = 0.5，无引用 = 0 |

**扣分项**：
- 包含"排毒""碱性体质"等伪科学概念：扣 3 分
- 推荐特定保健品或养生产品：扣 2 分
- 生活建议过于空泛（如"多喝水""注意身体"）：扣 1 分

---

## 维度三：可读性（10 分）

| 检查项 | 分值 | 评分标准 |
|--------|------|---------|
| 开头有季节感，让读者感受到"时令" | 2 分 | 时令感强 = 2，有但弱 = 1，无季节感 = 0 |
| 预防清单简洁可操作，读者可对照执行 | 2.5 分 | 简洁可操作 = 2.5，基本可操作 = 1.5，笼统 = 0.5 |
| 生活建议具体且有季节特色（非通用建议） | 2 分 | 有季节特色 = 2，通用建议 = 1，无 = 0 |
| 语气轻松友好，像朋友的善意提醒 | 2 分 | 友好自然 = 2，略生硬 = 1，说教式 = 0 |
| 高发疾病说明简明，不制造过度焦虑 | 1.5 分 | 简明平衡 = 1.5，有渲染 = 0.5，严重渲染 = 0 |

---

## 维度四：平台适配（10 分）

| 检查项 | 分值 | 评分标准 |
|--------|------|---------|
| 标题含季节/节气关键词 | 2 分 | 含关键词 = 2，模糊 = 1，标题党 = 0 |
| 预防清单使用列表格式，视觉清晰 | 2.5 分 | 列表规范 = 2.5，基本清晰 = 1.5，混乱 = 0 |
| 可考虑使用勾选框（- [ ]）视觉符号 | 1.5 分 | 有使用 = 1.5，未使用但清单清晰 = 1，无清单 = 0 |
| 字数在 600-1500 字区间 | 2 分 | 符合 = 2，略超 = 1，严重偏离 = 0 |
| 有配图建议且位置合理 | 2 分 | 合理 = 2，偏少 = 1，无 = 0 |

---

## 综合评分计算

```
总分 = 结构完整性 + 知识准确性 + 可读性 + 平台适配

32-40 分：优秀，可直接发布
24-31 分：合格，建议微调后发布
16-23 分：需修改，存在明显问题
0-15 分：需重写
```

## 季节性健康提示特别扣分项

- **文章无季节特殊性**（放之四季皆准的通用文章）：扣 3 分
- **生活建议过于空泛**（每条建议不足 20 字的描述）：扣 2 分
- **推荐养生伪科学**（"排毒""祛湿"等无科学依据的概念）：扣 3 分
- **高发疾病超过 5 种造成信息过载**：扣 1 分

## 一票否决项

以下任何一项出现，直接判定为"需重写"：
- 出现安全红线违规
- 包含养生伪科学（偏方、秘方、排毒疗法等）
- 推荐特定保健品或商业产品
- 高发疾病信息与当前季节完全不符
- 无任何引用标注
