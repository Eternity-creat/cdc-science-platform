# 案例故事文章质量评分标准

> 总分 40 分，四个维度各 10 分。
> 32 分及以上为优秀，24-31 分为合格，24 分以下需修改。

---

## 维度一：结构完整性（10 分）

基于蓝图要求的 5 个标准段落进行评估。

| 检查项 | 分值 | 评分标准 |
|--------|------|---------|
| 包含"故事引入"段且叙事有张力、吸引人 | 2.5 分 | 引人入胜、有悬念 = 2.5，有叙事但平淡 = 1.5，无叙事感 = 0 |
| 包含"病情分析"段且从案例自然过渡到医学解释 | 2 分 | 过渡自然、解释清晰 = 2，过渡生硬 = 1，无 = 0 |
| 包含"科普解读"段且从个案推广到一般知识 | 2.5 分 | 推广合理、知识系统 = 2.5，有推广但不够系统 = 1.5，仅停留在个案 = 0.5，无 = 0 |
| 包含"预防建议"段且与案例教训关联 | 2 分 | 关联紧密、可操作 = 2，有建议但关联弱 = 1，无 = 0 |
| 包含"结语"段且交代结局、升华主题 | 1 分 | 有交代 + 升华 = 1，有交代但平淡 = 0.5，无 = 0 |

---

## 维度二：知识准确性（10 分）

| 检查项 | 分值 | 评分标准 |
|--------|------|---------|
| 案例中的医学信息准确（症状、诊断、治疗方向） | 2.5 分 | 完全准确 = 2.5，有细微偏差 = 1.5，有明显错误 = 0 |
| 科普解读部分与当前医学共识一致 | 2.5 分 | 完全一致 = 2.5，有偏差 = 1.5，明显错误 = 0 |
| 预防建议有科学依据，未含未验证方法 | 2 分 | 全部有依据 = 2，含 1 条存疑 = 1，含伪科学 = 0 |
| 未夸大案例的代表性（除非有数据支撑） | 1.5 分 | 客观表述 = 1.5，有夸大倾向 = 0.5，严重夸大 = 0 |
| 引用标注规范 | 1.5 分 | 规范完整 = 1.5，有遗漏 = 0.5，无引用 = 0 |

**扣分项**：
- 案例细节中出现可识别的个人信息：扣 3 分（触发安全红线）
- 详细描述死亡/重症痛苦细节：扣 2 分
- 案例与科普部分脱节（故事归故事，科普归科普）：扣 2 分

---

## 维度三：可读性（10 分）

| 检查项 | 分值 | 评分标准 |
|--------|------|---------|
| 开头有叙事张力，让读者想继续读 | 2.5 分 | 张力强 = 2.5，有张力但弱 = 1.5，无张力 = 0 |
| 案例人物有温度（有细节、有情感），非冷冰冰的病例报告 | 2 分 | 有温度 = 2，偏冷 = 1，完全是病例报告 = 0 |
| 科普部分自然嵌入叙事，不突兀 | 2 分 | 嵌入自然 = 2，有衔接但生硬 = 1，完全割裂 = 0 |
| 语气有同理心，未将案例变成"反面教材" | 2 分 | 充满同理心 = 2，中立 = 1，有羞辱/批评倾向 = 0 |
| 结尾有升华，从个人经历推广到普遍意义 | 1.5 分 | 升华到位 = 1.5，有尝试但弱 = 0.5，无升华 = 0 |

---

## 维度四：平台适配（10 分）

| 检查项 | 分值 | 评分标准 |
|--------|------|---------|
| 标题含故事性元素（人物/事件），吸引点击 | 2 分 | 故事感强 = 2，有但弱 = 1，标题党 = 0 |
| 所有患者信息已脱敏（化名、无具体地点/单位） | 3 分 | 完全脱敏 = 3，有遗漏 = 1，未脱敏 = 0（触发红线） |
| 叙事段落与科普段落排版区分清晰 | 2 分 | 区分清晰 = 2，基本清晰 = 1，混乱 = 0 |
| 字数在 800-1500 字区间 | 1.5 分 | 符合 = 1.5，略超 = 1，严重偏离 = 0 |
| 有配图建议且位置合理（场景图 + 科普图） | 1.5 分 | 合理 = 1.5，偏少 = 0.5，无 = 0 |

---

## 综合评分计算

```
总分 = 结构完整性 + 知识准确性 + 可读性 + 平台适配

32-40 分：优秀，可直接发布
24-31 分：合格，建议微调后发布
16-23 分：需修改，存在明显问题
0-15 分：需重写
```

## 案例故事特别扣分项

- **患者信息未脱敏**：扣 3 分（且触发安全红线审查）
- **详细描述痛苦/死亡细节**：扣 2 分
- **将患者变成"反面教材"**（"都是因为不注意""自作自受"）：扣 2 分
- **故事与科普完全割裂**（前半段讲故事，后半段贴科普，无过渡）：扣 2 分
- **虚构案例未标注**：扣 1.5 分

## 一票否决项

以下任何一项出现，直接判定为"需重写"：
- 出现安全红线违规
- 患者信息未脱敏（出现真实姓名、住址、单位等可识别信息）
- 详细描述死亡/重症的痛苦细节
- 案例中的核心医学信息明显错误
- 贬低患者或将案例作为道德批判素材
- 虚构案例且未标注"根据多个真实情况综合整理"
