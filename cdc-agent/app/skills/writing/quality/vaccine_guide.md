# 疫苗接种指南文章质量评分标准

> 总分 40 分，四个维度各 10 分。
> 32 分及以上为优秀，24-31 分为合格，24 分以下需修改。

---

## 维度一：结构完整性（10 分）

基于蓝图要求的 7 个标准段落进行评估。

| 检查项 | 分值 | 评分标准 |
|--------|------|---------|
| 包含"引入"段且建立家长/读者共鸣 | 1.5 分 | 有共鸣感 = 1.5，有但生硬 = 0.5，无 = 0 |
| 包含"疫苗简介"段且通俗解释原理 | 1.5 分 | 有且通俗 = 1.5，有但学术化 = 0.5，无 = 0 |
| 包含"接种对象"段且区分推荐/慎用/禁忌 | 1.5 分 | 三级分类完整 = 1.5，仅列推荐人群 = 0.5，无 = 0 |
| 包含"接种程序"段且信息完整 | 1.5 分 | 含剂次、间隔、接种点 = 1.5，信息不全 = 0.5，无 = 0 |
| 包含"常见疑问"段（FAQ 形式，3-5 个 Q&A） | 2 分 | 3-5 个 Q&A 且覆盖核心疑虑 = 2，Q&A 少于 3 个 = 1，无 = 0 |
| 包含"注意事项"段 | 1 分 | 有且实用 = 1，有但笼统 = 0.5，无 = 0 |
| 包含"总结"段且给予信心 | 1 分 | 有且温暖 = 1，有但生硬 = 0.5，无 = 0 |

---

## 维度二：知识准确性（10 分）

| 检查项 | 分值 | 评分标准 |
|--------|------|---------|
| 疫苗类型、原理描述准确 | 2 分 | 完全准确 = 2，有细微偏差 = 1，明显错误 = 0 |
| 接种对象描述与官方指南一致 | 2 分 | 与最新指南一致 = 2，有遗漏 = 1，有错误 = 0 |
| 接种程序（剂次、间隔）准确 | 2 分 | 完全准确 = 2，有细微偏差 = 1，明显错误 = 0 |
| 不良反应描述客观，未夸大也未隐瞒 | 2 分 | 客观平衡 = 2，偏颇 = 1，严重失实 = 0 |
| 引用标注规范，关键信息有 {ref:N} | 2 分 | 规范完整 = 2，有遗漏 = 1，无引用 = 0 |

**扣分项**：
- 使用"绝对安全""零副作用"等绝对化表述：扣 2 分
- 推荐特定疫苗品牌：扣 3 分（且触发安全红线审查）
- FAQ 答案中出现未经证实的信息：扣 2 分

---

## 维度三：可读性（10 分）

| 检查项 | 分值 | 评分标准 |
|--------|------|---------|
| 开头理解家长焦虑，建立共情 | 2 分 | 共情到位 = 2，有尝试但不够 = 1，缺失 = 0 |
| 疫苗原理用类比或通俗方式解释 | 2 分 | 类比清晰 = 2，解释到位但无类比 = 1，晦涩 = 0 |
| FAQ 部分问答简洁，每个 Q&A ≤ 150 字 | 2 分 | 全部简洁 = 2，部分冗长 = 1，普遍冗长 = 0 |
| 未使用居高临下或说教语气 | 2 分 | 全程尊重 = 2，偶有说教 = 1，多处说教 = 0 |
| 接种程序信息清晰易查 | 2 分 | 列表/表格式呈现 = 2，文字堆砌 = 1，混乱 = 0 |

---

## 维度四：平台适配（10 分）

| 检查项 | 分值 | 评分标准 |
|--------|------|---------|
| 标题 14-30 字，含疫苗/接种关键词 | 2 分 | 完全符合 = 2，字数略超 = 1，标题党 = 0 |
| 摘要 80-120 字，回应核心疑问 | 1.5 分 | 完全符合 = 1.5，过长或过短 = 0.5，无 = 0 |
| FAQ 格式清晰，Q/A 分明 | 2 分 | 格式规范 = 2，基本清晰 = 1，混乱 = 0 |
| 有配图建议且位置合理 | 1.5 分 | 合理 = 1.5，偏少 = 0.5，无 = 0 |
| 字数在 600-1500 字区间 | 1.5 分 | 符合 = 1.5，略超 = 1，严重偏离 = 0 |
| 无营销用语、无品牌推荐、无 HTML | 1.5 分 | 全部合规 = 1.5，有 1 处 = 0.5，多处 = 0 |

---

## 综合评分计算

```
总分 = 结构完整性 + 知识准确性 + 可读性 + 平台适配

32-40 分：优秀，可直接发布
24-31 分：合格，建议微调后发布
16-23 分：需修改，存在明显问题
0-15 分：需重写
```

## 疫苗指南特别扣分项

以下情况在疫苗指南中特别严格：
- **贬低疫苗犹豫**：直接扣 3 分（违背共情原则）
- **隐瞒不良反应数据**：扣 3 分（违背信息完整原则）
- **使用恐吓性语言推动接种**：扣 2 分（违背 CDC 科普伦理）
- **未提及禁忌症**：扣 2 分（可能误导不适合接种的人群）

## 一票否决项

以下任何一项出现，直接判定为"需重写"：
- 出现安全红线违规（参见 universal_rules.md 第三节）
- 推荐特定疫苗品牌或厂商
- 疫苗安全性信息严重失实
- 禁忌症信息缺失或错误
- 核心接种程序信息与官方指南矛盾
