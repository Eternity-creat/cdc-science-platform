# 医务工作者受众画像

## 阅读特征

- **阅读场景**：工作间隙、下班后、值班室
- **阅读时长**：阅读时间碎片化，但对专业内容有较高的阅读耐心
- **阅读方式**：重点关注数据、指南更新、诊疗方案变化；对科普类内容可能觉得"太浅"
- **打开动机**：专业相关 + 最新指南 + 疫情动态 + 同行转发
- **信息偏好**：精准的流行病学数据、最新诊疗指南、防控措施更新

## 关注点排序

1. **最新的数据和指南更新**：有没有变化，变化了什么
2. **精准的流行病学信息**：发病率、病死率、传播动力学参数
3. **防控要点**：基层防控实操指引
4. **与其他疾病的鉴别**：诊断要点和鉴别诊断
5. **职业暴露防护**：院感防控、个人防护

## 语言偏好

- **喜欢的表达**：
  - 专业、精准、简洁
  - 使用医学术语但附带最新解读
  - 数据密集型呈现
  - 引用原始文献和指南

- **自称用语**：
  - "各位同道""各位同仁"
  - "一线医务工作者"
  - "基层防控人员"

- **偏好句式**：
  - "根据 XX 指南（2024 版）……"
  - "最新监测数据显示……"
  - "需注意以下几点：……"

## 需要回避的

- **过度简化**：把专业概念解释到"幼儿园水平"——他们不需要
- **居高临下的科普语气**："这个病你可能不了解……"
- **忽略专业读者的已有知识**：不需要解释什么是"飞沫传播"
- **空洞的口号式表述**：用数据和证据代替"请注意防护"
- **过时的指南信息**：必须标注版本号和发布时间

## 语气调整

- **基础语气**：专业 + 精准 + 尊重
- **调整方向**：
  - 比普通公众更加专业和严谨
  - 可以用专业术语，但标注最新指南的表述变化
  - 尊重读者的专业判断力
  - 突出"更新"和"变化"
  - 适当致敬一线工作者的辛苦

## 内容调整建议

- **开头**：直接说明"有什么更新/变化"，不铺垫
- **结构**：可以更加信息密集，减少过渡性文字
- **数据**：直接呈现原始数据，不需要过多通俗化翻译
- **引用**：详细标注文献来源，包括期刊名、卷号、页码
- **指南更新**：明确标注与上一版的变化点
- **结尾**：简洁，可以致敬一线工作者

## 典型疑问

> "最新版的诊疗方案有什么变化？"
> "当前的流行病学数据是什么？"
> "基层防控有什么新的建议？"
> "这个病的鉴别诊断要点是什么？"
> "职业暴露后的处置流程是什么？"

**写作时应预判这些疑问并在文中回答。重点突出"更新"和"变化"。**

## 特殊注意事项

- 医务工作者对 **数据准确性** 的要求最高，任何数据错误都会被同行质疑
- 涉及诊疗建议时，**必须标注指南版本和发布时间**
- 院感防控类文章，**需要区分不同科室/场景的差异化建议**
- 医务工作者是 CDC 公众号的 **重要传播节点**——他们看完会转发给患者和公众
- 可以设置 **"专业提示"** 段落，与面向公众的建议区分开
- 对于有争议的专业话题（如疫苗接种策略），**应引用国家级指南而非个人观点**
