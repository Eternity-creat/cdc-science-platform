# 微信公众号排版技巧卡

## 适用位置

整篇文章的排版和格式设计，确保文章在微信中的阅读体验最优。

---

## 技巧说明

微信公众号的阅读场景是手机屏幕，读者通常处于"滑动浏览"模式。好的排版能让读者：
- 快速抓住文章结构（扫读友好）
- 不会因密集文字而疲劳
- 在关键信息处停下来

核心原则：**让文章"看起来"就像"读起来"一样舒服。**

---

## 排版规则详解

### 规则一：段落节奏

**每段 2-4 行（手机屏幕行数）**，段落之间空一行。

**好的段落节奏：**
```
流感是由流感病毒引起的急性呼吸道传染病。

与普通感冒不同，流感起病更急，症状更重。

常见症状包括高热、头痛、肌肉酸痛和乏力。大多数人在 1-2 周内可以康复，但少数人可能出现严重并发症。
```

**差的段落节奏（过长）：**
```
流感是由流感病毒引起的急性呼吸道传染病，与普通感冒不同，流感起病更急，症状更重，常见症状包括高热、头痛、肌肉酸痛和乏力，大多数人在 1-2 周内可以康复，但少数人可能出现严重并发症，尤其是老年人、儿童和慢性病患者需要特别注意，建议及时就医。
```

### 规则二：视觉锚点

**每 300-400 字需要一个视觉锚点**，帮助读者在滑动中找到"停下来看"的理由。

视觉锚点类型：
- **小标题**（一、二、三 或 01、02、03）
- **加粗的关键句**
- **列表/清单**
- **引用块**（>）
- **配图**（[配图建议：描述]）

**示例：**
```
流感的传播主要依赖两个条件：病毒活性和接触频率。

**病毒在低温干燥的环境中更活跃**，这也是为什么冬季是流感高发季。

同时，冬季人们在室内聚集的时间增加，给病毒传播创造了更多机会{ref:1}。

[配图建议：冬季室内场景示意图，展示密闭空间中飞沫传播路径]

了解了传播条件，我们就可以有针对性地做好防护：

- 定期开窗通风，降低室内病毒浓度
- 在人员密集场所佩戴口罩
- 保持手部卫生，避免触摸口鼻
```

### 规则三：列表使用规范

列表是公众号文章中最高效的信息呈现方式之一。

**使用场景**：
- 并列的多条建议（≥3 条）
- 步骤/流程
- 对比要点

**格式规范**：
- 列表项之间不加标点
- 列表项以动词开头时保持一致（"勤洗手""戴口罩""多通风"）
- 列表前要有引导语

**好的列表示例：**
```
日常防护记住这五点：

1. 勤洗手，用流动水和肥皂，至少 20 秒
2. 在人员密集场所佩戴口罩
3. 每天开窗通风 2-3 次，每次 20-30 分钟
4. 咳嗽或打喷嚏时用肘部遮挡
5. 出现症状时居家休息，避免带病上班上学
```

**差的列表示例：**
```
- 勤洗手
- 戴口罩
- 通风
（太简略，缺乏可操作性）
```

### 规则四：引用与强调格式

**引用（>）**：用于标注权威来源的直接引述，或突出核心结论。

**加粗（**文字**）**：用于段落中最重要的 1-2 句话，帮助读者快速定位核心信息。

**示例：**
```
> "接种流感疫苗是预防流感最经济有效的手段。" ——中国疾控中心{ref:2}

每个流感季来临前，**60 岁以上老人、5 岁以下儿童、慢性病患者**应优先接种。
```

**注意**：
- 加粗不能滥用，每段最多加粗 1 处
- 不使用斜体（微信渲染效果差）
- 不使用下划线（与超链接混淆）

### 规则五：开头与结尾排版

**开头格式**：
```
[标题]

[1-2 句引言/钩子]

[空行]

[正文开始]
```

**结尾格式**：
```
[空行]

[总结/CTA 段落]

[空行]

---

参考文献：
[1] 来源名称. 标题. 日期
[2] 来源名称. 标题. 日期
```

## 字数与配图对照表

| 文章字数 | 建议配图数量 | 小标题数量 |
|---------|------------|-----------|
| 600-800 字 | 1-2 张 | 2-3 个 |
| 800-1200 字 | 2-3 张 | 3-5 个 |
| 1200-1500 字 | 3-4 张 | 4-6 个 |
| 1500-2000 字 | 4-5 张 | 5-7 个 |

---

## 选择规则

| 文章类型 | 重点排版技巧 |
|---------|------------|
| 疾病科普 | 小标题分段 + 症状列表 + 类比配图 |
| 疫苗指南 | FAQ 问答格式 + 接种程序列表 |
| 疫情提醒 | 数据加粗 + 时间标注 + 防护清单列表 |
| 谣言粉碎 | 引用块展示谣言 + 加粗结论 + 逐条列表 |
| 季节健康 | 检查清单（checklist） + 分类列表 |
| 案例故事 | 叙事段落 + 配图 + 科普插入段 |

---

## 禁止

1. **禁止使用 HTML 标签**：微信编辑器不支持，会显示为乱码
2. **禁止使用脚注语法**：[^1] 在微信中无法渲染
3. **禁止大段文字无分段**：超过 5 行的段落必须拆分
4. **禁止全篇加粗或大面积加粗**：加粗失去重点效果
5. **禁止列表项超过 7 条**：过长列表应分组或精简
6. **禁止配图无说明**：每张图都要有 [配图建议] 说明
7. **禁止在正文中使用表格**：微信对 Markdown 表格支持差，用列表替代
8. **禁止段落首行缩进**：微信排版中首行缩进反而影响阅读
